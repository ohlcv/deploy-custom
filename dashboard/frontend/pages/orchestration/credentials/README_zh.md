### 描述

此页面允许您管理交易所凭证：

- 查看所有可用账户及其关联的凭证
- 创建新的交易账户
- 删除现有账户
- 向账户添加新的交易所API凭证
- 从账户中移除凭证

您在此处存储的凭证将被Hummingbot机器人用于连接交易所并执行交易。所有敏感信息都会被安全存储和管理。
